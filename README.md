## 加入欢迎功能

用户进入机器人所在频道时自动发送频道欢迎，进入服务器默认频道时发送全局服务器欢迎。可在网页设置中独立开关。

**配置方法**
1. WebUI → 设置 → 机器人 Profile → **加入欢迎**，打开 toggle 开关
2. 重启机器人使配置生效
3. 给机器人 ServerAdmin 权限（TeamSpeak 客户端中右键机器人 → ServerGroup → 加入 Server Admin），否则无法收到进出频道事件

**功能说明**
- **频道欢迎**：用户进入机器人所在频道时，发送 `欢迎{昵称}加入{频道名}，!help 获取播放指令，玩的开心哦`
- **服务器欢迎**：用户进入服务器默认频道时，发送 `欢迎{昵称}加入{服务器名}！使用 !ai 与我对话聊天哦`
- **防打扰**：机器人切换频道后 3 秒内不发送欢迎
- **开关独立**：每个机器人可独立开启/关闭

## AI 对话控制

通过 `!ai <内容>` 与 DeepSeek AI 对话，AI 可自动识别意图并执行播放命令。

**配置**
1. 从 [platform.deepseek.com](https://platform.deepseek.com) 注册获取 API Key
2. WebUI → 设置 → DeepSeek API Key 填入保存

**使用示例**
| 命令 | 效果 |
|------|------|
| `!ai 你好` | AI 回复问候 |
| `!ai 播放周杰伦的歌` | AI 解析 → 自动执行播放（自动切 QQ 音乐） |
| `!ai 下一首` | AI 解析 → 执行下一曲 |
| `!ai 怎么播放歌曲` | AI 文字回答使用方法 |

**AI 记忆功能**
- AI 可用 `[MEM]...[/MEM]` 标签记住你的偏好和习惯
- 下次对话时自动加载已记住的信息
- WebUI → 设置 → AI 记忆，可一键清空

**智能音源切换**
- 默认使用网易云音乐，加 `-q` 切换 QQ 音乐
- 周杰伦、王力宏、S.H.E 的歌曲自动切 QQ 音乐（网易云无版权）

## 播放队列可逆切歌

- `!prev` 回到上一首后，`!next` 优先回到原来的位置，不会跳到随机歌
- 支持 Random / RandomLoop 模式

## 播放错误诊断

播放失败时自动检测平台登录状态和 VIP 状态，给出具体原因和解决提示，而非简单显示"Cannot play"。
